# cmdSave
A small utility for saving and runing commands.

### Command syntax:

##### add a command
-a commandName "yourcommand -bal -bla --balbla -blafile /home/bla/bla/blabal.txt"

##### run a command
-r commandName

### Important note
** DO NOT ** delete the cmds.txt file! it contains all of the commands that you saved!
